# 엑셀 함수의 기능과 형식
### 엑셀의 채우기 기능
- 연속된 숫자나 일정한 규칙으로 변화하는 숫자들을 쉽게 입력할 수 있도록 함

### 엑셀 함수 기능과 형식
- 기능: 값, 셀 참조, 함수 등을 사용하여 새로운 값 생성
- 형식: =함수이름(인수)
  - =SUM(A1:A3)

### 함수 사용의 규칙
- 수식은 등호로 시작
- 함수 이름 다음에 인수를 묶는 괄호 필요
- 인수는 숫자, 셀 범위, 논리값, 문자값, 다른 함수도 가능
- 인수가 여러개일때는 , 사용

# 날짜/시간 함수 활용
### 날짜/시간 함수의 예
- DATE(year, month, day)
  - 특정 날짜의 일련번호를 구해주는 기능
- YEAR(날짜 또는 숫자)
  - 날짜에 해당하는 연도를 표시
- NOW()
  - 현재시점을 나타내는 기능으로 인수가 필요없는 함수
